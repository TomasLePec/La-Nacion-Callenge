# La-Nacion-Callenge
App for La Nacion:
 - Front-End: Next + TypeScript
 - Back-End: Node + Express + TypeScript

## Run Project
You have two options to run the project:
 - Inside the two directories and follow their README.md
 - Run all at once:
    - npm run install
    - npm run generate
    - npm run start
